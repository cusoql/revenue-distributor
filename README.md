# Revenue Distributor

Простое приложение на Python для распределения доходов по различным категориям.

## 📌 Описание

Revenue Distributor - это небольшой проект для домашних пользователей, который помогает разделить доход на несколько частей, таких как сбережения, расходы, инвестиции или другие личные категории.

Проект был создан для отработки основ Python, расчетов и простой программной логики.

## Features Особенности

- Распределение доходов по категориям
- Простая и понятная логика Python
- Легко модифицируется под личные нужды
- Полезный практический проект для начинающих

## Technologies Технологии

- Питон
